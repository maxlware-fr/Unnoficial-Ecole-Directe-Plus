# Unnoficial Ecole Directe Plus (UEDP)

***U*EDP** est une application desktop codé sous *electron* pour naviguer sur **Ecole Directe Plus** *(EDP)*.
*EDP* augmente *EcoleDirecte*, avec une interface moderne et intuitive, enrichie de fonctionnalités exclusives, le tout de façon gratuite, libre et open-source.

***U*EDP** est une version non-officielle développé par **Maxlware**, développeur chez *EDP*.

## Installation
0: Les prérequis sont **Node.js 24.11.1** ou plus.
1: Télécharger le git
```
git clone https://github.com/maxlware-fr/Unnoficial-Ecole-Directe-Plus.git
```
2: Aller sur le dossier du logiciel
```
cd C:/Unnoficial-Ecole-Directe-Plus/src
```
3: Télécharger les dépendances
```
npm install
```
4: Installer l'extension [***U*EDP Support**](https://github.com/maxlware-fr/Unnoficial-Ecole-Directe-Plus/tree/main/src/Unnoficial-Support-Ecole-Directe-Plus-Unblock).
5: Lancer le logiciel
```
npm start
```
